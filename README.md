# Home Repair and Technician Booking System
### Software Requirements Engineering Project FAST NUCES

## Project Overview
A platform that connects homeowners with verified local 
technicians such as electricians, plumbers, and painters 
for booking, scheduling, and emergency repair services.

## Project Phases

| Phase | Title | Status |
|-------|-------|--------|
| Phase 1 | Project Idea and Scope | Completed |
| Phase 2 | Requirement Elicitation | Completed |
| Phase 3 | Requirements Analysis and Modeling | Completed |

## Key Deliverables
- 32 Functional Requirements and 10 Non-Functional Requirements
- 42x42 Requirements Interaction Matrix
- MoSCoW Prioritization Table covering all 42 requirements
- Use Case Diagram with 25 use cases and 3 actors
- 28 User Stories for Customer, Technician, and Admin
- Context Diagram and Level 0 DFD using Gane and Sarson notation
